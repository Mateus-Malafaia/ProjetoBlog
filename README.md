ProjetoBlog

Um blog simples e funcional desenvolvido com Node.js, Express.js e EJS, permitindo criar, visualizar, editar e excluir posts.
Este projeto foi criado com foco em aprendizado, prática de rotas, views dinâmicas e estruturação de projetos web.

Funcionalidades

Criar posts com título e conteúdo

Listar posts na página inicial

Editar posts existentes

Excluir posts

Layout responsivo e estilizado

Renderização dinâmica usando EJS

Organização do projeto em rotas, views e arquivos estáticos

Posts mantidos em memória (sem banco de dados)

Tecnologias utilizadas

Node.js

Express.js

EJS (Embedded JavaScript Templates)

CSS Responsivo

JavaScript (Frontend e Backend)
